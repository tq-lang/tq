# Security Policy

## Supported Versions

Security fixes are applied to the latest released version.

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Use GitHub Security Advisories to report privately:

1. Go to `https://github.com/tq-lang/tq/security/advisories/new`
2. Submit details, impact, and a minimal reproduction
3. Include the affected version and environment

If you cannot use GitHub Security Advisories, open a normal issue and clearly
state that you need a private contact channel for a security report.

## Response Expectations

- Initial acknowledgment target: within 5 business days
- We will validate, assess severity, and coordinate a fix and disclosure plan
- We will credit reporters in release notes unless anonymity is requested
